Blenxy

- Presenting Blenxy! a Blender 2.79b application template that aims to be like a comfort zone for SMG modeling (“Blen” for Blender and “xy” from Galaxy). It isn’t much but honest work.


Features:

- Environment Lightning already turned on (for material viewing).

- A custom splash screen that totally helps in the modeling process!

- Default measurement unit scaled down to 0.005 (1/200 - metric) so you don’t have to scale up the models before exporting (models will have a good size on Whitehole/Game), you can have an idea of the size they will have on SMG. The idea is to have a controlled environment in which you can tell the dimensions of everything you are making. I wasn’t able to put a name to the custom unit as I don’t know how to write python scripts so, for the record, I baptized it with the name “Galaxy Unit” or “GU” for short. Each GU is 1/200 of a metric unit (pretty cool huh).

- A Mario model file “mario.fbx” is included in the ZIP file of the project template so you can test your models against a real Mario model to see if you need to scale the model up or down upon making! (it will show everytime Blenxy is loaded like the default Blender cube).

- XY visual grid extended to 200 lines (in case you want to create big planets).

- Z axis is visible (just for visual reference).

- Clip start is very low (10cm) and clip end very high (2km) (it is like how much you can zoom down or back without the objects on scene disappearing - basically because objects on SMG can be either too tiny or too big).
