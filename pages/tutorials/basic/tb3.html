Extract ARC file contents

Tools needed:

Refer to the "Tools" section for download.

- Arc Convertor

- Wine (for Linux only)

For Windows:

- Among the Arc Convertor files there is a folder named “res”, inside it there is a file called “arcExtract.exe” (Fig. 1).

(Fig. 1 - "arcExtract.exe" file location)

- Drag and drop that ARC file you want to extract into the “arcExtract.exe” file and the extracted files will appear in the same place the ARC file is (Fig. 2).

(Fig. 2 - Extracted files from ARC file)

For Linux:

- As Linux does not execute EXE files natively you have to install Wine to be able to use the Arc Convertor tool.

- With Wine and the “arcExtract.exe” file from the “res” folder of the Arc Convertor tool, open a terminal and execute the following command (Fig. 3):

wine path/to/arcExtract.exe path/to/file.arc

(Fig. 3 - Command to extract contents from ARC file)

- All files within the ARC file will be extracted on the same path the ARC file is (Fig. 4).

(Fig. 4 - Extracted files from ARC file)

Extract BMD/BDL file contents

Tools needed:

Refer to the "Tools" section for download.

- SuperBMD

- Wine (for Linux only)

For Windows:

- Among the SuperBMD files there is a file named “SuperBMD.exe” (Fig. 5).

(Fig. 5 - "SuperBMD.exe" file location)

- Drag and drop the BMD/BDL file into the “SuperBMD.exe” file and the extracted files will appear in the same place the BMD/BDL file is (Fig. 6).

(Fig. 6 - Extracted files from BDL file)

For Linux:

- As Linux does not execute EXE files natively you have to install Wine to be able to use SuperBMD.

- With Wine and the “SuperBMD.exe” file among the SuperBMD files, open a terminal and execute the following command (Fig. 7):

wine path/to/SuperBMD.exe path/to/bmd-or-bdl-file

(Fig. 7 - Command to extract contents from BMD/BDL file)

- All files within the BMD/BDL file will be extracted on the same path the BMD/BDL file is (Fig. 8).

(Fig. 8 - Extracted files from BDL file)

Note: for some reason SuperBMD didn't work with an absolute path, I got it working with a relative one as the one shown on Fig. 8.
