Prerequisites:

- Have setup j3dview --> Link

- Know how to convert PNG into BTI --> Link

Steps:

- Open j3dview and load the BMD/BDL model you want (Fig. 1).

(Fig. 1 - Penguin BDL model opened on j3dview)

- Expand the texture's desplegable menu at the left of the program and select the texture to replace (Fig. 2).

(Fig. 2 - BDL Model texture selected on j3dview)

- Right-click on the texture and click on “Replace”. Then, select the location of BTI file you want to replace (Fig. 3). After finishing replacing textures, remember to save the model (File > Save Model).

(Fig. 3 - BDL Model texture replaced on j3dview)

Note 0: remember to use textures that have dimensions that can be expressed by the power of 2, otherwise, Whitehole won't load the model.
