Tools needed

Refer to the "Tools" section for download.

- Wiimm's SZS Toolset (WST)

Steps:

This process is mostly used when replacing textures on BMD/BDL models with j3dview.


Convert PNG to BTI

- Open a CMD window in the same place the PNG file you want to convert is (Fig. 1).

(Fig. 1)

- Type and enter the following command: wimgt copy “<input PNG file>” “<output BTI file>” (Fig. 2).

(Fig. 2)

- After doing that a BTI file should create on the same folder as the PNG file (Fig. 3).

(Fig. 3)

Note: BTI files are always greater in size in relation to the PNG file from were it was created because of how mipmaps work.

- Alternatively, you can create a BATCH file containing the following code (you can name it “png-to-bti.bat” for example):

wimgt copy %1 "%~dp1%~n1.bti"

pause

- Then, just drag and drop the PNG file you want to convert into the BATCH file and a BTI file should create on the same place the PNG file dragged and dropped is.


Convert BTI to PNG

- Open a CMD window in the same place the BTI file you want to convert is (Fig. 4).

(Fig. 4)

- Type and enter the following command: wimgt copy “<input BTI file>” “<output PNG file> --no-mm” (Fig. 5).

(Fig. 5)

- After doing that a PNG file should create on the same folder as the BTI file (Fig. 3).

(Fig. 6)

Note 1: BTI files extracted from j3dview can’t be converted/processed by WST’s wimgt for some reason. Other tool I tried to convert a j3dview exported BTI file said that the file was corrupted (maybe j3dview does not export correctly BTI files?).

Note 2: the "--no-mm" flag is used to only get the base image used in the BTI file rather than all images within it (mipmaps).

- Alternatively, you can create a BATCH file containing the following code (you can name it “bti-to-png.bat” for example):

wimgt copy %1 "%~dp1%~n1.png" --no-mm

pause

- Then, just drag and drop the BTI file you want to convert into the BATCH file and a PNG file should create on the same place the BTI file dragged and dropped is.
